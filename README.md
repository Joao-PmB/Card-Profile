# Card Profile

Card de perfil simples com links para redes sociais, desenvolvido com HTML e CSS, além de manipulação DOM com JavaScript para mudar o tema da página para claro ou escuro.

O projeto foi criado com o objetivo de praticar CSS moderno e manipulação do DOM com JavaScript. É um cartão de identificação digital simples.

Durante o desenvolvimento, foram aplicados conceitos de Flexbox, efeitos visuais (hover) e localStorage para persistir a preferência de tema do usuário.

## 📸 Demonstração

Acesse o projeto:
🔗 https://card-profile-tan.vercel.app/

## 🚀 Funcionalidades

- Card de perfil minimalista.
- Links para redes sociais (GitHub, LinkedIn, WhatsApp, Instagram).
- Alternância entre tema claro e escuro.
- Persistência de tema com localStorage.
- Efeitos hover suaves.
